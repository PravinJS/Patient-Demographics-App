Patient Details App

Install below

- [Mongo](https://www.mongodb.com/) 4.0.4+
- [ExpressJS](https://expressjs.com/) 4.16.3+
- [ReactJS](https://reactjs.org/) 16.5.0+
- [Node](https://nodejs.org/en/) 11.4.0+ (It's recommended to use 10.15.1 LTS)

Steps to run this application

```
$ cd patients-app
$ cd server
$ yarn install
$ nodemon index.js
```

```
$ cd patients-app
$ cd client
$ yarn install
$ yarn start
```
